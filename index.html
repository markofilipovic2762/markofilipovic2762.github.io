About me

I am a Full-stack Web Developer with 5 years experience from Serbia. I create successful websites that are fast, easy to use, and built with best practices. My main experience is in frontend and backend development, but I also have a passion for design. Given that producing a modern website requires the combination of design, server technology, and the layer that users interacts with, I believe having experience in both design and development allows for making the most optimal user experiences.

  • Frontend Development

    React library is my expertise. For more complex project, i use Redux and React Router. When project does not need React, Vanilla javascript enters the scene :)

  • Backend Development

    For backend side I use most Node.js with Express framework, with REST api and MongoDB for databases. I know, as well, how to work with SQL and GraphQL.

  • UI UX Design

    CSS and its frameworks like SASS, Bootstrap and Material UI. For some special design i like to use Abobe XD, but its just a hobby, for now.
